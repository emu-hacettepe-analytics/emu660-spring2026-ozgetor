Summary: The participants’ academic backgrounds and how they transitioned from Industrial Engineering to Data Science were discussed. Question 1: What is the biggest advantage of industrial engineers in data science?

It is their ability to think “systemically” and to manage data not only by analyzing it, but also as a decision-making and optimization process.

  2. Having a very large amount of data.

# function
compute_stats <- function(x) {
  list(
    mean = mean(x, na.rm = TRUE),
    median = median(x, na.rm = TRUE),
    variance = var(x, na.rm = TRUE),
    iqr = IQR(x, na.rm = TRUE),
    min = min(x, na.rm = TRUE),
    max = max(x, na.rm = TRUE)
  )
}
